- Menampilkan Main Activity dan Fragment-nya
    - Memastikan fragment MOVIES dapat ditampilkan.
    - Memastikan recycler view fragment MOVIES dapat di-scroll.
    - Memastikan fragment TV SHOWS dapat ditampilkan.
    - Memastikan recycler view fragment TV SHOWS dapat di-scroll.

- Menampilkan Detail Activity
    - Memastikan salah satu item fragment MOVIES dapat diklik.
    - Memastikan item MOVIE dapat tampil di Detail Activity
    - Memastikan data item MOVIE benar / cocok.
    - Memastikan tombol Favorite bisa ditekan untuk menambahkan item ke daftar Favorite Movies.
    - Memastikan tombol back di toolbar dapat diklik.
    - Memastikan fragment TV SHOWS dapat ditampilkan.
    - Memastikan salah satu item fragment TV SHOWS dapat diklik.
    - Memastikan item Tv Show dapat tampil di Detail Activity.
    - Memastikan data TV SHOWS benar / cocok.
    - Memastikan tombol Favorite dapat ditekan untuk menambahkan item ke Favorite Tv Shows.
    - Memastikan tombol back di toolbar dapat diklik.

- Menampilkan Favorite Activity
    - Memastikan fragment movies di Favorite Activity bisa ditampilkan.
    - Memastikan item Movie (yang telah ditambahkan ke Favorite) dapat diklik.
    - Memastikan item Movie dapat tampil di Detail Activity.
    - Meastikan tombol Favorite dapat ditekan untuk menghapus item dari daftar Favorite Movies.
    - Memastikan tombol back di toolbar dapat diklik.
    - Memastikan fragment TV SHOWS di Favorite Activity dapat ditampilkan.
    - Memastikan item Tv Show (yang telah ditambahkan ke Favorite) dapat diklik.
    - Memastikan item Tv Show dapat tampil di Detail Activity.
    - Meastikan tombol Favorite dapat ditekan untuk menghapus item dari daftar Favorite Tv Shows
    - Memastikan tombol back di toolbar dapat diklik untuk kembali ke Favorite Activity.
    - Memastikan tombol back di toolbar dapat diklik untuk kembali ke Main Activity.
